# 한줄광고 출력 위젯 (plusad widget)

한줄광고 모듈에 등록된 광고를 전광판(롤링) 형태로 출력하는 라이믹스(Rhymix) 위젯입니다.

![미리보기](preview.png)

---

## 요구 사항

- [라이믹스(Rhymix)](https://rhymix.org)
  - 한줄광고 모듈이 XE를 지원하지 않으므로 XE 환경에서는 사용하실 수 없습니다.
- [한줄광고(plusad) 모듈](https://github.com/Lastorder-DC/rx-module-plusad)이 설치되어 있어야 합니다.

## 설치 방법

위젯 파일을 아래 경로에 배치하면 설치가 완료됩니다.

```
./widgets/plusadWidget/
```

별도의 설치 과정 없이 파일을 지정 폴더에 복사하는 것만으로 사용할 수 있습니다.

## 위젯 설정

관리자 페이지에서 위젯을 추가할 때 아래 항목을 설정할 수 있습니다.

| 설정 항목 | 설명 | 기본값 |
|-----------|------|--------|
| 모듈 이름 | 한줄광고 모듈의 이름 | `plusad` |
| 광고 제목 제한 | 광고 내용의 최대 표시 글자 수 | `20` |
| 광고사인 표시 | 광고 영역에 'AD' 표시 여부 (표시함 / 표시 안함) | - |
| 닉네임 출력여부 | 광고 등록자 닉네임 표시 여부 (표시함 / 표시 안함) | - |
| 링크 연결 방식 | 새창 연결 / 바로 연결 | - |
| 스크롤 속도 | 롤링 속도 (숫자가 높을수록 빠름) | `50` |
| 스크롤 간격 | 광고와 광고 사이의 간격 (단위: ms, 1000 = 1초) | `1000` |

## 스킨

| 스킨 이름 | 설명 |
|-----------|------|
| default | 기본 스킨 (검은 글씨) |
| white | 기본 스킨 (흰색 글씨) |
